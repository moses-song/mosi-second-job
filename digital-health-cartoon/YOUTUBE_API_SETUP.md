# YouTube Data API v3 활성화 방법

## 단계별 가이드

### 1. Google Cloud Console 접속
- https://console.cloud.google.com/ 접속
- Google 계정으로 로그인

### 2. 프로젝트 선택/생성
- 기존 프로젝트 선택 또는 새 프로젝트 생성
- 현재 프로젝트 ID: 620793672820

### 3. YouTube Data API v3 활성화
- 좌측 메뉴 → API 및 서비스 → 라이브러리
- "YouTube Data API v3" 검색
- 클릭 후 "사용 설정" 버튼 클릭

### 4. API 키 생성 (필요시)
- 좌측 메뉴 → API 및 서비스 → 사용자 인증 정보
- "+ 사용자 인증 정보 만들기" → API 키
- API 키 복사하여 보관

### 5. 할당량 확인
- YouTube Data API v3는 일일 10,000 단위 무료
- 1검색 = 100 단위 (일일 100회 검색 가능)

### 6. 제한 설정 (권장)
- API 키에 YouTube API만 허용하도록 제한
- 특정 도메인이나 IP 주소에서만 사용하도록 제한
