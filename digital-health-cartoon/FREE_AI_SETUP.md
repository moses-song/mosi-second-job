# 무료 AI API 사용 방법

## Google Gemini API (무료)
- [Google AI Studio](https://aistudio.google.com/app/apikey)에서 무료 API 키 발급
- 월 15회 요청까지 무료
- GPT-4 수준의 성능

## Hugging Face API (무료)
- [Hugging Face](https://huggingface.co/settings/tokens)에서 토큰 발급
- 다양한 모델 사용 가능
- 제한적이지만 무료

## 설정 방법
1. Google AI Studio 방문
2. API 키 생성
3. Cloudflare Workers 환경 변수에 GEMINI_API_KEY 추가
4. 배포 완료
